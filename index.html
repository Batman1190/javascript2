🎯 Lesson 1: Say Hello to JavaScript!

Hey there, young coder! 👋 JavaScript makes websites come alive!

Let's learn console.log() - it makes your computer talk!

📝 Example:

console.log("Hello, I'm learning!");
Words inside " " are what your computer will say!

🎨 Lesson 2: Magic Boxes (Variables)!

Variables are magic boxes 📦 that store stuff!

Use let to create them!

📝 Example:

let myName = "Luna"; let myAge = 7; console.log("Hi! I'm " + myName);
"let" creates a box, "=" puts something in it!

🎯 Exercise 1: Your Info!

🔢 Lesson 3: Numbers Game!

📝 Example:

🎪 Lesson 4: Making Decisions!

if helps make choices! 🤔

IF this is true, THEN do this!

📝 Example:

let weather = "sunny"; if (weather === "sunny") { console.log("Go to park! ☀️"); }
Use === to check if things are equal!

🎵 Lesson 5: Loops - Do it Again!

Loops repeat things! 🔄 No more typing same code!

📝 Example:

for (let i = 1; i <= 5; i++) { console.log("Count: " + i); }
This counts from 1 to 5!
